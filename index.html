Q1: How does javascript work?

জাভাস্ক্রিপ্ট একটি client-side scripting language।যা web-browser এর ভিতরে client side এ চলে। প্রতিটি পপুলার web-browser এর নিজস্ব জাভাস্ক্রিপ্ট ইঞ্জিন রয়েছে।যখন browser এ জাভাস্ক্রিপ্ট কোড রান করে তখন ইঞ্জিন 'parser'দ্বারা syntatic error না পাইলে,AST নামের data structure তৈরি করে,কোডটিকে মেশিন কোডে রুপান্তরিত করে সিস্টেম এ পাঠানো হয়। এবং পরিশেষে byte-code টি সিস্টেম বা ইঞ্জিন দ্বারা run হয়।

Q2: What does javascript event loop do?

জাভাস্ক্রিপ্ট single thread এ সব operation execute করে। ইভেন্ট লুপের কাজ হচ্ছে এটি ক্রমাগত চেক করে যে Call stack খালি কিনা, খালি থাকলে ক্রমাগত event queue থেকে প্রথম ইভেন্টটি নিয়ে এটিকে কল স্ট্যাকে push করে দিবে এবং ঠিক মতো execute করবে। খালি না থাকলে current processed হওয়া function গুলি execute হইতে থাকে। এবং প্রতিটি function execute হওয়ার পর stack থেকে pop করে দেয়।

Q3: difference between local storage and session storage?

    Local Storage:
  • Local Storage এর data নতুন ট্যাব ওপেন করা অথবা Browser বন্ধ করে পুনরায় খোলা হলেও data সেভ হয়ে থাকে।
  • কোন ‍নির্দিষ্ট ট্যাব এর জন্য কাজ করে না।
  • Local Storage এ data যেভাবে set করা হয় (localStorage.setItem('mango',50))
  • Data নিজ হাতে delete করতে হয়।
    Session Storage:
  • Session Storage এর data যতক্ষণ ট্যাব বা ব্রাউজার খোলা থাকে ততক্ষণ স্থায়ী হয়,এইগুলো বন্ধ হলে data চলে যাবে।
  • যে কোন নির্দিষ্ট ট্যাব এর জন্য কাজ করে।ওই ট্যাব বন্ধ করলে data চলে যায়
  • Session Storage এ data যেভাবে set করা হয় (sessionStorage.setItem('mango',50))
  • ব্রাউজার অথবা ট্যাব বন্ধ করলে data মুছে যায়।
